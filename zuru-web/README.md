## Cómo arrancar el proyecto

### Desarrollo
1. Instala las dependencias:
   ```bash
   npm install
   ```
2. Arranca el servidor de desarrollo:
   ```bash
   npm run dev
   ```

### Producción
1. Compila el proyecto:
   ```bash
   npm run build
   ```
2. Sirve los archivos estáticos generados en `dist/` con tu servidor favorito.
